# site-mvp

Projeto de site estático com páginas de apresentação, produtos, sobre e contato.

## Estrutura de arquivos

- `index.html` — página principal
- `produtos.html` — página de produtos
- `sobre.html` — página sobre a marca
- `contato.html` — página de contato com formulário e mapa
- `css/style.css` — estilos compartilhados
- `img/` — imagens do projeto

## Como usar

1. Abra `index.html` no navegador.
2. Navegue entre as páginas usando os links de menu.
3. A página `contato.html` inclui um formulário e um mapa embutido do Google Maps.

## Boas práticas aplicadas

- organização de arquivos em pastas semânticas (`css/`, `img/`)
- HTML com links relativos para os recursos
- CSS centralizado em um arquivo único para facilitar manutenção
- validação de formulário no HTML para melhor experiência do usuário

## Observação

Se for publicar no GitHub Pages, mantenha esse mesmo layout de pastas e use o diretório raiz como site.
